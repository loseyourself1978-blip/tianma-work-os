Original mission seed

The demo is grounded in the real 2026WC project request.

The source request asks TWOS to analyze the full 2026 World Cup schedule, team quality, player condition, style, tactics, history, upset risk, and mainstream bookmaker odds, then predict each next Beijing-time match day only.

Prediction output

Win/draw/loss, score, total goals, recommendation, upset watch, and confidence or priority.

Review loop

Review yesterday actual results vs predictions, record hits and misses, update calibration, then prepare the next-day forecast.

Schedule base

Use a 104-match Beijing-time schedule base for the 48-team, 12-group format and knockout path.

What the system is trying to understand

Interpret the goal before assigning work.

Task type

Prediction preparation, review, and learning from outcomes.

Likely project type

2026WC prediction and review project with recurring manual check-ins.

Cadence assumption

Requested review rhythm: daily BJT/SGT 20:30 from 2026-06-10 to 2026-07-20.

Clarification notes

Later TWOS should ask which odds sources, injury notes, team news, and scoring rules the user wants to supply.

Task decomposition

The team breaks the 2026WC mission into workstreams.

Grounded demo workflow only. Schedule data, odds, injuries, team news, outcomes, and feedback stay user-supplied in this static prototype.

Schedule Base

Build and maintain a 104-match Beijing-time schedule, group stage plus knockout structure, match-day windows, and next-day prediction scope.

Team Intelligence

Track team level, player form, injuries or rotation when supplied, tactics, formation, pressing, transition, set pieces, club-season load, and key-player condition.

Odds / Market Layer

Review mainstream bookmaker odds, 1X2, handicap, totals, score odds when available, market consensus, and suspicious pricing.

Prediction Layer

Produce win/draw/loss, score, total goals, recommendation or main pick, upset watch, confidence, and priority.

Daily Review Loop

Compare previous actual result vs predicted result, score hit/miss, total goals hit/miss, upset miss/hit, and calibration changes before next prediction.

Multi-Role Decision

Use 赛果复盘员, 赔率盘口分析师, 战术风格分析师, 球员状态分析师, 历史与大赛心理分析师, 爆冷审计员, and 总指挥.

Output / Sync

Generate a daily report, compact sync, feedback capture, and next-day prediction focus.
