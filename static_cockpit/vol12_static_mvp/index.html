First-run workflow

From pasted sync to scoped Codex task

  1. Paste sync or task.User provides the state snapshot.
  2. TWOS auto-forms an AI Board.Mission roles are selected from static mock slots.
  3. Board identifies risk, noise, and forbidden scope.Noise and permission boundaries are separated from decisions.
  4. Board locks Active Next.One next checkpoint becomes visible.
  5. Codex receives scoped task only if approved.Repo work stays explicit and bounded.
  6. Result returns as validation receipt.Evidence is checked against the requested scope.
  7. DUXD converts mistakes into guardrails.Operating errors become future constraints.
