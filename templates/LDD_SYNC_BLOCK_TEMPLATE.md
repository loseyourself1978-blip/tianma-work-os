# LDD Sync Block Template v0.1

## Standard LDD Sync Block

```text
LDD → Tianma Work OS Sync Block

1. Review time:
2. Review type:
3. Data sources:
4. Account updates:
5. Confirmed actions:
6. Market conclusions:
7. Crypto conclusions:
8. U.S. equity conclusions:
9. LDD strategy status:
10. Historical-position status:
11. Robot/grid-bot status:
12. Pending actions:
13. Risk warnings:
14. Forecast review result:
15. Review score:
16. DUXD / product feedback:
17. New or updated memory rules:
18. Tianma Work OS design updates:
19. Documents to update:
20. Items to sync into GitHub:
```

## Usage

At the end of every LDD review, the assistant should generate a completed sync block automatically. The user copies the completed block into Tianma Work OS.
